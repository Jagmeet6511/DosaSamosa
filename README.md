# Dosa Samosa Cafe Website

Official website for Dosa Samosa Cafe.

## Features
- Modern responsive design
- Menu section
- Contact and location information
- Fast loading website

## Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS

## Run Locally

Install dependencies:

npm install

Start development server:

npm run dev

Open in browser:

http://localhost:8080
